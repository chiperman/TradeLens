# TradeLens - 产品需求文档 (PRD) v0.2

## 1. 产品概述

**TradeLens** 是一款专为加密货币波段交易者设计的盈亏计算与资产追踪工具。它不仅提供基于实时行情的双场景（已成交 vs 实时预测）保本价与利润试算，还能通过连接交易所 API 自动拉取真实交易记录，沉淀为历史账本，并支持跨平台多端访问（Web + Desktop）。

## 2. 核心定位

- **精准测算**：扣除双向手续费后的真实盈亏与保本价。
- **自动化记账**：解放双手，通过 API Key 自动同步交易所订单流水。
- **智能聚合**：将同一基础资产（Base Asset）下的不同计价货币（如 USDT、USDC、FDUSD）统一折算合并，按基础资产（如 BTC、ETH）进行全局盈亏追踪分析。

## 3. 技术栈选型 (全栈 Web + 桌面端)

- **前端框架**：Next.js (React) + Tailwind CSS + shadcn/ui (现代化的精美 UI)
- **后端/数据库**：Next.js API Routes + Supabase (PostgreSQL，用于存储用户配置、历史账本，并加密存储 API Keys)
- **桌面端化方案**：**Tauri** (使用 Rust 打包 Web 页面为轻量级跨平台桌面应用，性能优于 Electron，且完美兼容 Next.js)
- **外部接口**：Binance API (或其他主流交易所 API) + 免费实时汇率 API

## 4. 核心功能模块

### 4.1 交易测算器 (Calculator)

- **实时行情**：对接交易所 API 获取最新资产价格和法币汇率。
- **双场景对比测算**：
  - **Scenario A (历史复盘/已成交)**：输入真实的买入和卖出价，计算已实现的净利润、法币等值收益和真实手续费磨损。
  - **Scenario B (实时预测/下单前试算)**：输入目标本金与当前市价，计算保本价（Break-even）、预测盈亏、当前价差，指导何时挂单。
- **动态参数**：支持自定义 Maker/Taker 手续费率、自定义法币汇率计算。

### 4.2 自动化历史账本 (Ledger)

- **API 自动拉取**：用户绑定交易所只读 API Key，系统定时或手动触发同步历史订单。
- **智能合并与分类**：
  - **基础资产聚合**：系统识别 `BTC/USDT`、`BTC/USDC`、`BTC/FDUSD` 等，统一归类到 `BTC` 资产下。
  - **盈亏对齐**：自动匹配买入与卖出动作，计算单笔波段的真实盈亏。
- **手动补录**：支持手动添加未接入 API 的记录或链上交易。

### 4.3 数据透视看板 (Dashboard)

- **全局资产统计**：总投入、总净利、总手续费消耗、整体胜率。
- **单币种表现**：各币种（如 BTC, SOL）的独立盈亏报告。

### 4.4 数据导入与导出 (Data I/O)

- **导出**：一键导出所有交易流水为 `.xlsx` (Excel) 或 `.json`。
- **导入**：支持按照模板从 Excel/CSV 导入历史数据。

## 5. 项目路线图 (Phases)

- **Phase 1 (基础版)**：搭建 Next.js 框架，还原并优化“波段双场景计算器”逻辑。
- **Phase 2 (账本与数据库)**：接入 Supabase，实现用户登录、保存计算历史。
- **Phase 3 (自动化与聚合)**：接入交易所 API，实现订单自动同步与基础资产（BTC/USDT 等）聚合逻辑。
- **Phase 4 (桌面端)**：集成 Tauri，构建 macOS/Windows 原生应用。
