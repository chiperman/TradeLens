## 1. 产品概述

**TradeLens** 是一款面向个人投资者的全资产类别投资追踪与分析平台。V2 从 V1 的「加密货币波段计算器」全面演进为覆盖 **美股、港股、美股期权、加密货币** 的综合投资管理工具。

### 1.1 核心使命

- **全资产追踪**：统一管理跨市场、跨交易所的投资记录
- **精准分析**：扣除手续费后的真实盈亏、多维度收益率计算、基准对比
- **智能洞察**：丰富的可视化图表、波段保本价试算、期权 Payoff 分析
- **自动化**：API 自动同步交易记录，减少手工录入

### 1.2 目标用户

- 个人投资者（当前为单用户系统，登录即为管理员）
- 同时参与美股/港股/加密货币市场的活跃交易者

### 1.3 不做的事情（Scope Out）

- ❌ 实盘交易下单（纯分析平台）
- ❌ A 股、新加坡股市场
- ❌ 多腿期权策略（只支持单腿 Put/Call）
- ❌ 桌面端 Tauri 打包（推迟到 V3）
- ❌ PDF 投资报告导出（推迟到 V3）
- ❌ 社交/分享功能

## 2. 支持的资产类别

| 资产类别 | 市场 | 数据来源 | 支持的操作 |
|---------|------|---------|-----------|
| 美股 (US Equity) | NYSE, NASDAQ | Longbridge OpenAPI | 实时行情、交易记录同步、分红/拆股 |
| 港股 (HK Equity) | HKEX | Longbridge OpenAPI | 实时行情、交易记录同步、分红/拆股 |
| 美股期权 (US Options) | CBOE | Longbridge OpenAPI | 期权记录、Greeks 追踪、Payoff 图 |
| 加密货币 (Crypto) | 全球 | Binance / Bitget / OKX API | 实时行情、交易记录同步 |

### 2.1 美股期权支持详情

- **主要交易模式**：卖 Put（Sell Put）+ 平仓 Put（Buy to Close）
- **追踪字段**：
  - 标的资产（Underlying）
  - 期权类型（Call / Put）
  - 行权价（Strike Price）
  - 到期日（Expiration Date）
  - 权利金（Premium）
  - 合约数量
  - Greeks（Delta, Gamma, Theta, Vega）
- **自动处理**：
  - 期权到期作废 → 自动关闭持仓、确认权利金收入
  - 期权被行权 → 自动生成对应股票买入/卖出记录，更新持仓
- **可视化**：基于行权价和权利金的 Payoff Diagram

## 4. 功能模块

### 4.1 Dashboard（总览仪表板）

> 登录后的首页，提供全局资产概览

**核心信息**：
- 总资产净值（USD 为主 + CNY 折算）
- 当日 / 本周 / 本月盈亏
- 持仓分布（按资产类别：美股 / 港股 / Crypto / 期权）
- 关键 KPI 卡片：
  - 总收益率
  - 胜率（Win Rate）
  - 盈亏比（Profit/Loss Ratio）
  - 最大回撤（Max Drawdown）
  - 夏普比率（Sharpe Ratio）
- 净值曲线（vs S&P 500 / QQQ 基准线）
- 最近交易动态

### 4.2 Portfolio（持仓管理）

#### 4.2.1 持仓列表

- 按资产类别分 Tab：**股票** / **期权** / **加密货币**
- 每个持仓显示：
  - 标的名称 / 代码
  - 持仓数量
  - 平均成本
  - 当前市价（实时）
  - 未实现盈亏（金额 + 百分比）
  - 所属交易所 / 券商
- 支持「整合视图」和「分交易所视图」切换：
  - **整合视图**：跨交易所同一标的合并计算（如 Binance + OKX 的 BTC 合并）
  - **分交易所视图**：按交易所独立展示

#### 4.2.2 持仓详情

点击某个持仓进入详情页：
- 标的基本信息
- 历史交易记录（该标的的所有买卖）
- 成本 vs 市价对比图
- 收益走势图
- 波段分析（保本价计算器联动）

#### 4.2.3 实时行情

- 持仓标的自动订阅实时价格推送
- 股票通过 Longbridge WebSocket
- Crypto 通过各交易所 WebSocket
- 刷新频率：股票随行情推送，Crypto 实时

### 4.3 Calculator（波段计算器）

> 继承 V1 的核心功能并大幅扩展

#### 4.3.1 股票/Crypto 波段计算

- **Scenario A（历史复盘）**：输入已成交的买入/卖出价，计算已实现 P&L
- **Scenario B（实时预测）**：基于实时市价，计算：
  - 保本价（Break-even Price，扣除手续费后）
  - 在目标价格买入/卖出的预期 P&L
  - 当前市价下的 P&L
- **多价位 P&L 试算**：输入多个价位，计算每个价位的盈亏情况
- **手续费模型**：
  - 全局预设费率（在 Settings 中配置）
  - 单笔交易可覆盖费率

#### 4.3.2 手续费模型

| 市场 | 费用项 | 默认值 | 可自定义 |
|------|--------|-------|---------|
| 美股 | 佣金 | 按 Longbridge 费率 | ✅ |
| 港股 | 佣金 + 印花税(0.13%) + 交收费 + 交易费 + 证监会征费 | 按 Longbridge 费率 | ✅ |
| Crypto | Maker/Taker 费率 | 0.1% | ✅ |
| 美股期权 | 佣金 + 合约费 | 按 Longbridge 费率 | ✅ |

#### 4.3.3 期权 Payoff 分析

- 输入：行权价、权利金、合约数量、期权类型（Call/Put）、买/卖方向
- 输出：
  - 保本价
  - Payoff Diagram（到期日 P&L 图）
  - 最大盈利 / 最大亏损
  - 当前市价下的理论价值

### 4.4 Ledger（交易账本）

#### 4.4.1 交易记录

- **数据来源**：
  - API 自动同步（标记为 `auto`）
  - 手动录入（标记为 `manual`）
  - CSV/Excel 导入（标记为 `import`）
- **去重机制**：
  - API 同步时，若检测到与已有手动记录匹配（基于标的、时间、价格、数量的模糊匹配），标记为「疑似重复」
  - 用户在专门的「去重审核」界面手动确认：合并 / 保留 / 删除
- **记录字段**：
  - 交易时间
  - 标的代码 / 名称
  - 交易方向（买入 / 卖出）
  - 成交价格
  - 成交数量
  - 手续费
  - 交易所 / 券商
  - 数据来源（auto / manual / import）
  - 备注（可选）

#### 4.4.2 资金流水（入金/出金）

- 按交易所/券商分别记录
- 字段：
  - 时间
  - 金额
  - 货币类型
  - 交易所/券商
  - 方向（入金 / 出金）
  - 备注（可选）

#### 4.4.3 Corporate Actions（公司行为）

- 自动处理（通过 Longbridge API 获取数据）：
  - **现金分红**：记录分红金额，计入收益
  - **股票分红**：更新持仓数量
  - **拆股/合股**：按比例调整持仓数量和成本均价

#### 4.4.4 数据导入

| 来源 | 格式 | 说明 |
|------|------|------|
| 手动模板 | CSV / Excel | 提供标准模板下载 |
| Longbridge | API 自动同步 | OAuth 2.0 认证 |
| Binance | API 自动同步 | HMAC 签名认证 |
| Bitget | API 自动同步 | HMAC 签名认证 |
| OKX | API 自动同步 | HMAC 签名认证 |

#### 4.4.5 数据导出

- 支持格式：CSV / Excel / JSON
- 导出范围：可按时间范围、资产类别、交易所筛选
- 可用于数据备份

### 4.5 Analytics（分析报表）

#### 4.5.1 收益率计算

| 类型 | 说明 | 用途 |
|------|------|------|
| 简单收益率 | (结束值 - 开始值) / 开始值 | 快速概览 |
| 时间加权收益率 (TWR) | 消除资金流入流出影响 | 评估投资决策质量 |
| 金额加权收益率 (MWR/IRR) | 考虑资金流入流出时机 | 评估实际赚了多少 |

#### 4.5.2 基准对比

- S&P 500 指数
- QQQ (NASDAQ-100 ETF)
- 支持在同一图表中叠加显示组合收益率 vs 基准收益率

#### 4.5.3 图表库

> 所有图表支持时间维度切换：日 / 周 / 月 / 季度 / YTD / 年 / 全部

| 图表 | 描述 | 优先级 |
|------|------|--------|
| 累计 P&L 曲线 | 净值随时间变化 | P0 |
| 资产配置饼图 | 按类别/标的的持仓占比 | P0 |
| 月度/年度收益柱状图 | 按月/年的收益汇总 | P0 |
| 收益率对比折线图 | 组合 vs S&P 500 / QQQ | P0 |
| 单标的收益走势图 | 某只股票/Crypto 的 P&L 历史 | P1 |
| 回撤曲线 (Drawdown) | 从峰值下跌的百分比 | P1 |
| 持仓成本 vs 市价对比图 | 柱状图对比每个持仓的成本和现价 | P1 |
| 日历热力图 | 类似 GitHub 贡献图的每日盈亏 | P1 |
| 收益率瀑布图 | 各标的对总收益的贡献分解 | P2 |
| 期权 Payoff Diagram | 到期日损益曲线 | P1 |
| 波段价差雷达图 | 多维度波段分析 | P2 |
| 资金流入/流出趋势图 | 入金/出金随时间变化 | P2 |
| 手续费消耗分析图 | 累计手续费按时间/交易所分布 | P2 |

### 4.6 Settings（设置）

#### 4.6.1 账户设置

- 邮箱/密码管理
- 已关联的 OAuth 账户

#### 4.6.2 API Keys 管理

| 平台 | 认证方式 | 说明 |
|------|---------|------|
| Longbridge | OAuth 2.0 | 推荐方式 |
| Binance | API Key + Secret | 只读权限 |
| Bitget | API Key + Secret + Passphrase | 只读权限 |
| OKX | API Key + Secret + Passphrase | 只读权限 |

#### 4.6.3 手续费率配置

#### 4.6.4 通知配置（Bark）

- Bark 服务器地址
- Device Key

  - 总资产涨跌幅超过阈值（如 ±3%）
  - 当日收益率超过阈值
  - 自动同步完成通知

#### 4.6.5 偏好设置

- 基准货币：USD（主）/ CNY（辅）
- 主题切换（多主题支持）
- 语言切换（中文 / English）

## 6. 通知系统

### 6.1 Bark 推送

- **服务**：Bark（iOS 推送服务）
- **触发条件**：
  - 投资组合日 P&L 超过用户设定的阈值
  - 总收益率达到设定目标
  - 数据同步完成/失败通知
- **不触发的场景**：
  - 单笔交易的价格提醒（不做）
  - 个股涨跌幅提醒（不做）
- **配置项**：
  - Bark Server URL
  - Device Key
  - 阈值设置（涨幅 %、跌幅 %）
  - 开关控制每种通知类型

## 7. 数据安全

### 7.1 认证与授权

- Supabase Auth（邮箱/密码 + GitHub OAuth）
- 行级安全策略 (RLS)：所有表启用，确保用户只能访问自己的数据

### 7.2 API Key 存储

- Longbridge：使用 OAuth 2.0 Token，存储 refresh_token（加密）
- Crypto 交易所：API Key + Secret 加密存储在 Supabase
- 所有 API Key 仅申请 **只读权限**

### 7.3 数据备份

- 用户可手动导出全部数据（CSV/Excel/JSON）
- 无自动云备份（依赖 Supabase 自身备份机制）
